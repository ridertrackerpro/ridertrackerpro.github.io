Chapter 11 ‚Äî Backup & Restore

Rider Tracker Pro stores all data in your browser's localStorage. While this is reliable for normal use, it can be cleared if you clear your browser data, reinstall the app, or change devices. The backup system lets you export and import all your data as a JSON file.

Automatic Backup Prompt

After every time you add or delete a shift, refuel, or extra distance entry, the app may display a backup suggestion popup: 'Your data has been modified. Do you want to download a backup?' You can choose DOWNLOAD to save a backup immediately, or NOT NOW to dismiss.

Manual Backup Export

Open the Monthly Report (SUMMARY button) and tap EXPORT BACKUP. A file named backup_YYYY-MM-DD.json will be downloaded to your device. Keep this file safe ‚Äî it contains the complete database of all your data.

Restore from Backup (Import)

Open the Monthly Report and tap IMPORT BACKUP. Select the .json backup file from your device. The app will load all data from the file and reload. All existing data will be replaced by the backup content.

‚ö†Ô∏è Warning: Importing a backup completely replaces all current data. Make sure you are importing the correct file. Consider exporting a new backup first before importing an old one.

End-of-Month Notification

On the 1st day of each new month, when you open the app, a notification popup will appear reminding you that the month has ended and asking if you want to print the report. This is a good time to also export a backup.

Chapter 12 ‚Äî Reset Database (RESET DB)

The RESET DB button is located at the far right of the bottom navigation bar, displayed as a miniature safe with a lock icon. This feature is protected by a double-tap mechanism to prevent accidental deletion.

How to Reset

To reset the database:

  1. Tap the safe icon once ‚Äî the safe door will animate open and the lock will unlock.

  2. Immediately (within 1 second) tap it again ‚Äî a confirmation dialog will appear.

  3. Tap OK to confirm. ALL data will be permanently deleted and the app will reload.

‚ö†Ô∏è IRREVERSIBLE: A database reset deletes all shifts, refuels, extra distances, targets, and tax settings. This action CANNOT be undone. Always export a backup before resetting.

Chapter 13 ‚Äî Language Selection

Rider Tracker Pro supports two languages: Italian (üáÆüáπ) and English (üá∫üá∏). The language flags are displayed at the top of the main screen.

Tapping a flag instantly switches all interface text, labels, buttons, alerts, and report headers to that language. The active language flag appears at full opacity; the inactive one is dimmed.

Each language has its own independent settings (units, currency, etc.) stored separately. This means you can have different unit preferences for each language, though it is recommended to keep them consistent.

The last selected language is saved automatically and will be restored when you reopen the app.
